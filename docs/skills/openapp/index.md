## Introduction

This skill open a application when a button is pressed. You can bind different applications to different buttons.

Put your full application path in the `APP` variable in the config tab.

Make sure you have started the PC Controller service in the host application. Once the service is started, you will find the `AW agent` in the list of available devices.

## Hardware

- A button module
